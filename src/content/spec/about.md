# About


>此博客主要用于记录个人学习、工作和生活中的一些思考和经验。
>不进行任何形式的商业使用，仅用于个人学习。
博客的内容主要包括但不限于：
>- 技术博客：记录个人学习的技术知识、项目经验和解决问题的方法。
>- 生活博客：分享个人的生活经验、爱好和思考。
>- 工作博客：记录个人在工作中的思考和经验。

本项目基于 [Astro](https://astro.build/) 构建，Astro 是一个由 With Astro 团队维护的开源项目，遵循 MIT 许可证。
- **源码仓库**：`https://github.com/withastro/astro`
